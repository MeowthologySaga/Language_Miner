# Recipe: Roguelite Action

## 적합한 아이디어

- 뱀서류
- 탄막 생존
- 아레나 액션
- 짧은 런 반복 게임

## 핵심 루프

```txt
전투 시작
→ 적 처치
→ 경험치/재화 획득
→ 무기/스킬 선택
→ 보스 또는 시간 목표 도전
→ 런 종료 보상
→ 장기 해금
```

## 다이아 소비처

- 캐릭터 해금
- 캐릭터/스킨 뽑기
- 시작 유물 선택권
- 부활
- 도전 입장권
- 펫/동료 뽑기

## 필수 시스템

- 이동/공격
- 적 스폰
- 레벨업 선택지
- 런 종료 보상
- 장기 해금
- 저장
- 다이아 소비 요청

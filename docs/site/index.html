01 · THE GAP

뜻은 아는데, 필요한 순간에는 왜 말이 안 나올까요?

단어를 알아보는 기억과 문장을 직접 꺼내는 기억은 다릅니다. Language Miner는 공부를 “봤다”에서 끝내지 않습니다.

단어만 외울 때

의미는 알아도 문장이 흩어집니다

어떤 전치사와 동사가 자연스러운지, 실제 상황에서 어떤 말투를 쓰는지 함께 남지 않습니다.

문장으로 외울 때

어휘·문법·상황이 한 덩어리로 남습니다

내가 발견한 실제 문장을 카드로 만들면 다음 대화에서 꺼낼 단위가 생깁니다.

02–03 · BALANCE

인풋으로 발견하고, 아웃풋으로 내 것으로.

많이 읽고 듣는 것과 직접 말하고 쓰는 것을 같은 문장으로 연결합니다.

읽고 듣기 · Input

자연스러운 영어를 알아보는 힘

PDF, 웹, 듣기, 영상에서 내가 정말 쓰고 싶은 표현을 발견합니다. 읽기 카드와 듣기 카드로 형태와 소리에 익숙해집니다.

말하고 쓰기 · Output

필요한 순간에 꺼내는 힘

말하기 카드, 영작, 캐릭터챗에서 뜻과 상황만 보고 영어를 만들어 봅니다. 틀린 부분은 다음 복습의 힌트가 됩니다.

04 · ONE SENTENCE

한 문장이 실제 사용까지 가는 길

I’m running a little late. 하나로 학습 루프를 끝까지 돌아봅니다.

발견

읽던 글에서 약속에 늦을 때 쓰고 싶은 문장을 선택합니다.

카드

먼저 웹 리더에서 읽기 카드를 만듭니다. 필요하면 듣기 카드와 말하기 카드는 각각 듣기·Life Mining 화면에서 더합니다.

복습

영어를 알아본 뒤, “조금 늦을 것 같아요”를 보고 직접 문장을 말합니다.

사용

영작이나 약속 상황의 캐릭터 대화에서 같은 표현을 다시 씁니다.

05 · PRACTICE WORLDS

표현은 대화에서, 학습 보상은 게임에서

캐릭터챗은 배운 표현을 대화에서 연습하고, PlayZone은 학습으로 번 다이아를 게임 기능에 쓰게 해 학습 동기를 이어 줍니다.

Character Chat

대화 상대와 상황을 골라 연습

배운 표현을 캐릭터와의 약속, 여행, 일상 대화 안에서 사용합니다. 클라우드 AI를 연결하면 전송할 캐릭터 설정과 최근 대화 범위를 먼저 확인합니다.

PlayZone

학습으로 번 다이아를 사용하는 검증형 UGC 게임

어비스 서머너, 드릴하트 디펜스, 고양이 오디세이 같은 공식 게임과 사용자 게임팩에서 학습으로 번 다이아를 사용할 수 있습니다. 팩은 기술 검사 후 격리해 실행하고, 실행 가능한 상태만 열며 네트워크는 기본 거부합니다.

06 · FIRST START

AI 설정 없이 따라 하는 10분 시작 순서

새 설치의 기본값은 AI 미연결입니다. 카드와 복습을 먼저 경험하고 필요한 기능만 나중에 연결하세요.

  1. 웹 리더의 내장 연습 글을 엽니다.
    외부 접속 없이 고정 예문으로 실제 카드 저장을 연습합니다.
  2. 읽기 카드의 원문·뜻·문맥을 확인합니다.
    내가 기억할 상황에 맞게 직접 고칩니다.
  3. 저장하고 첫 복습을 합니다.
    답 보기를 누르기 전에 먼저 떠올립니다.
  4. 짧은 영작에서 다시 꺼냅니다.
    같은 뜻을 보고 영어 문장을 직접 입력합니다.
  5. 필요할 때만 AI를 연결합니다.
    로컬 Ollama를 먼저 살펴보고, 클라우드는 전송 내용과 비용을 확인합니다.
07 · YOUR BOUNDARY

내 데이터, 내 키, 내가 고르는 연결.

“로컬”과 “무료”를 막연하게 말하지 않고 실제 저장·전송·비용 경계를 보여 줍니다.

  • 로컬 저장카드와 대화는 기본적으로 이 PC에 저장됩니다. API 키 외 학습 데이터는 평문일 수 있습니다.
  • AI 미연결 기본값동의 전 Gemini·Google AI 요청은 없습니다. loopback Ollama만 로컬로 표시합니다.
  • BYOK 클라우드본인의 키를 쓰고, 공급자·전송 내용·호출 수·비용 범위를 작업 전에 확인합니다.
  • 비용 가드의 한계앱 수치는 로컬 예상 사용량 가드입니다. 공급자 청구를 막으려면 quota와 예산 알림이 필요합니다.
  • UGC 실행 제한해시·권한·CSP·압축 구조를 검사하고 준비 상태가 아닌 팩은 실행하지 않습니다.
  • 중앙 추적 없음개발자 소유 광고, 분석, 텔레메트리 서버를 사용하지 않습니다.

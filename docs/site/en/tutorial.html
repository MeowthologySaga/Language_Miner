01 · Install and first launch

Choose your languages first. AI can wait.

  1. Use the installer on a PC you will keep using. Choose the portable build to try the app without an installation wizard. If you are unsure, use the installer.
  2. This first beta is unsigned, so Windows may show a warning. Do not run a file from an uncertain source; use the SmartScreen and checksum guide when needed. Portable may still use the Windows user-data folder.
  3. On first launch, choose App language: English, your Native language, and Learning language: English, then choose Next.
  4. Read the sentence-card and input/output introduction. On the last AI screen, choose Explore the app.
Why three language settings? App language controls menus, Native language controls meanings and explanations, and Learning language controls cards and practice. You can change them later in Settings.

02 · Remember a usable sentence

Keep a complete unit you can retrieve, not only a definition.

An isolated word

late = after the expected time

You know the meaning, but not necessarily the natural verb, tone, or sentence to send a friend.

A sentence in context

I’m running a little late.

The phrase run late, an in-progress situation, and a natural tone stay together.

Reading and listening build recognition. Speaking and writing build retrieval from a meaning or situation. Practicing both directions turns “I have seen this” into “I can use this.”

03 · Your first learning loop

Sentence card → Add card → Review → Write

You only need four places in the sidebar for your first loop.

Read & Listen · Input → Web Reader → Review → Speak & Write · Output → Writing Practice

To rehearse the controls first, use Manage → Tutorial → Reading Cards → Web Reader. It is a sandbox copy: tutorial progress is recorded, but your real card library is unchanged.

1) Find a sentence in Web Reader

  1. Open Read & Listen · Input → Web Reader.
  2. Choose Language Miner practice text on the home screen. It is built into the app and does not contact an external website.
  3. Select running a little late, then choose Sentence card in the selection tool.
This sentence may also be a default sample. A new profile can already contain a sample card with the same sentence. Saving from the practice text may leave two similar cards; hide or delete the one you do not need afterward.

2) Check the reading card and add it

Review the source sentence, natural meaning, selected expression, and nearby context. You can edit generated content. Remove names, contact details, work information, or context you do not want to keep, then choose Add card.

3) Recall before revealing the answer

  1. Open Review, then choose Start review for the reading-card deck.
  2. Look at the front and recall the meaning and situation.
  3. Choose Show answer, then rate the real effort as Again, Hard, Good, or Easy.
A rating is not a grade. It tells the scheduler when to show the card again. If you only remembered after revealing the answer, a lower rating may help more.

4) Retrieve the same sentence in writing

  1. Open Speak & Write · Output → Writing Practice.
  2. Use the prompt about telling a friend you will arrive a little late, and type I’m running a little late. yourself.
  3. Choose Show answer to compare your response with the suggestion. Choose Rewrite to try again.

5) Use it in Character Chat — optional

Character Chat is optional and requires an AI connection. Open Speak & Write · Output → Character Chat, choose a character and Practice in learning language, then send the same sentence. With cloud AI, first review the notice that the character definition, recent conversation, your message, and card hints may be transferred.

04 · Three card types

Make the card that matches how you got stuck.

CardCreated inWhat you do on the front
Reading cardWeb Reader, Document ReaderSee English; recall meaning and structure
Listening cardListening Loop, Video ReaderHear the source segment; recognize the sentence
Speaking cardLife MiningSee a native-language situation; produce the learning language

These cards come from different input experiences; they are not all generated on one screen. Add a listening card when text is easy but sound is not. Add a speaking card for something you often want to say.

05 · Use your own material

Choose study material where you were already reading or listening.

Document Reader

Open a supported document, select an expression, and make a reading card. Return through bookmarks and recent documents.

Listening Loop

Repeat a transcript segment, mark hard-to-hear sound chunks, and save a listening card.

Video Reader

Check a subtitle or transcript segment and loop range, then save the current segment as a listening card.

Life Mining

Collect native-language expressions you genuinely wanted to say as speaking-card candidates. Automatic capture starts off.

Media and backups Cards and transcript records can be backed up, but source PDFs, audio, video, and local paths are excluded. After restoring on another PC, you may need to select the source again.

06 · PlayZone

A reason to return after study.

Character Chat tests an expression in conversation. PlayZone lets you spend locally earned diamonds on optional game features such as summons, revives, and rerolls, reinforcing motivation to keep studying. It is not an in-game language-learning mode. To keep the app smaller, even official games may download the first time you use them.

  1. Open PlayZone and choose Abyss Summoner, Drillheart Defense, or Cat Odyssey.
  2. Choose Install and play.
  3. Review download size, creator, version, source, license, SHA-256, and storage, network, and diamond permissions.
  4. Choose Review and download. After technical checks produce a runnable state, choose Play.
What UGC checks do not prove The app checks file structure, integrity, and requested permissions. It does not guarantee content suitability, copyright status, or every creator claim. Check the source and license yourself.

07 · Connect AI

Connect only the feature you need, after checking its boundary.

Manage → Settings → AI & Voice → Card generation

Disconnected

The default on a new install. Start with card management, review, and other features that do not need AI.

Ollama

Use a separately installed local model. Only loopback hosts such as localhost, 127.0.0.1, and ::1 are labeled local.

Gemini · Google

Bring your own key. Review provider, model, transferred content, estimated calls, and estimated cost before a task.

ChatGPT Web

Manually copy a prompt and paste back the JSON result. The app does not read your login cookies or ChatGPT page.

08 · Backup and deletion

Back up learning records, not keys or source media.

Manage → Settings → Sync & Data → Full backup and restore

  • Can include: cards, reviews, profiles, Life Mining, listening records, character conversations, routines and missions, and PlayZone saves.
  • Excluded: API keys, login cookies, original local paths, logs, caches, models, and source PDFs, audio, and video.

Choose Create backup file to save a .lembackup. During restore, check the checksum, version, and conflicts, then preview Restore as new profiles, Merge non-conflicting data, or Replace current data.

09 · Browser extension beta

Reduce the allowed sites and context before capture.

The first public beta uses optional manual installation rather than the Chrome Web Store. Download Language-Miner-Extension-<version>.zip and SHA256SUMS.txt from the same official Release, verify the hash, and extract the ZIP. Open chrome://extensions, enable Developer mode, choose Load unpacked, and select the folder that directly contains manifest.json.

  • Choose allowed sites and whether to collect only your own messages.
  • Set surrounding context, maximum length, and masking rules.
  • Selecting text alone sends nothing. The extension sends the expression and nearby context to the local app only after you run the sentence-card action.
  • Inspect and clear failed items in the offline pending queue.

10 · When something does not work

Look for the state and next action before copying technical details.

Ollama will not connect
Check installation, running state, downloaded model name, and loopback URL, then choose Start and connect Ollama again.
A game pack keeps loading
Check the official game’s install state and pre-run security report. Expand and copy technical details only when filing an issue.
Media does not open after restore
Source PDF, audio, video, and paths are not backed up. Select the source file again.
SmartScreen appears
Do not run the file if you cannot verify both the official Release and SHA-256.

What next?

Make one more reading card, or save a listening card for a sentence you miss in real audio. Useful material matters more than collecting a large number of cards.

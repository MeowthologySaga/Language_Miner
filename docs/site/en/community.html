02 · INSTALL

Open what you downloaded

No unzipping and no folder hunting. Import a game as its .lemgame file or a character card as its .json file.

PLAYZONE GAME
  1. 1

    Select Download .lemgame.

  2. 2

    Open PlayZone in the app.

  3. 3

    Select Add LEM file and choose the download.

  4. 4

    Review the security report and select Confirm and install.

CHARACTER CARD
  1. 1

    Download the character JSON.

  2. 2

    Open Speak & Write · Output → Character Chat.

  3. 3

    Select Import JSON.

  4. 4

    Review its source and select Review and import.

Even easier: choose Install and play for any official game in PlayZone. The app downloads and verifies the same pinned release.

Share a character card

Share a data-only JSON exported by the app. It cannot run HTML or JavaScript, and the limit is 2 MiB.

  1. Use Export as Language Miner pack
  2. Upload to a GitHub Release or public Google Drive link
  3. Enter the title, description, link, and license

04 · MODERATION

Spam and attacks do not enter the catalog directly

A review boundary separates public GitHub submissions from the catalog, while this static site has no login or upload server.

01

GitHub sign-in

GitHub manages accounts, reports, and blocking.

02

Rate and duplicate limits

More than three submissions in 24 hours or a repeated distribution link is closed and locked.

03

No listing before approval

Posts are public but do not enter this catalog before review.

04

Edits trigger re-review

Changing an approved link or content removes approval.

05

No submitted-file execution

The anti-spam check never downloads or executes user files.

06

Static site

No upload, database, or shared API key reduces attack and server-cost exposure.

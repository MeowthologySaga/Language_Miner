01 · WHAT TO SHARE

Content you can share now

The catalog starts with the two formats the app can actually import.

Character Chat

Character chat card or pack

A single JSON created with Export as Language Miner pack. It contains character settings and safe image information without HTML or JavaScript.

Recommended: GitHub Release · simpler option: public Google Drive
PlayZone

Game Pack

A .lem or .lemgame file. Publish its version, SHA-256, license, and requested permissions. Only a pack that passes the app validator can run.

Recommended: a GitHub Release that pins source and artifact
Proposal

Propose another compatible type

Choose Other compatible format proposal in the form. A format the app cannot import yet is reviewed for support first.

A proposal does not automatically become installable content.
02 · THREE STEPS

Simple enough for a first share

Upload the file, register its link, and follow the review state.

  1. 1

    Upload your file to a public link

    GitHub Releases are safest and easiest to version. If that is unfamiliar, set a Google Drive link so anyone with the link can view.

  2. 2

    Register the link and rights

    Add the description, creator, license, and permissions such as remote images, network, storage, or diamond use. Game Packs also need the filename and SHA-256.

  3. 3

    Follow the review state

    New entries first appear in the unreviewed queue. Only entries marked ugc-ready appear in the reviewed catalog.

04 · SHARE SAFELY

Check before you share

The official sharing space accepts general-audience content with clear publication rights.

  • Your rightsYou may publish and redistribute every included piece of code, art, audio, and text.
  • No secretsYou removed API keys, tokens, email addresses, private conversations, local paths, and logs.
  • Declare permissionsYou listed every external connection, storage action, card action, and diamond action.
  • External linksGoogle Drive and similar sources are marked as mutable, unverified external links.

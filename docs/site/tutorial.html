01 · 설치와 첫 실행

처음에는 언어만 정하고, AI는 건너뛰어도 됩니다.

  1. 공식 GitHub Release에서 설치판 또는 포터블판과 같은 버전의 SHA-256 파일을 받습니다.
  2. 계속 사용할 PC라면 설치판을, 설치 과정 없이 시험하려면 포터블판을 고릅니다. 포터블판도 학습 데이터는 Windows 사용자 데이터 위치에 저장할 수 있습니다.
  3. 첫 실행 화면에서 앱 언어: 한국어, 모국어: 한국어, 학습 언어: English를 고르고 다음을 누릅니다.
  4. 문장 카드와 인풋·아웃풋 안내를 읽습니다. 마지막 AI 화면에서는 앱 둘러보기를 선택합니다.
왜 언어가 세 개인가요? 앱 언어는 버튼과 메뉴, 모국어는 뜻과 설명, 학습 언어는 카드와 연습에 사용할 언어입니다. 설정에서 나중에 바꿀 수 있습니다.

02 · 문장으로 기억하기

단어 뜻이 아니라, 실제로 꺼낼 한 덩어리를 남깁니다.

단어만 외우기

late = 늦은

뜻은 알지만 어떤 동사와 말투로 친구에게 알려야 할지는 남지 않을 수 있습니다.

문장으로 기억하기

I’m running a little late.

run late라는 표현, 진행 중인 상황과 자연스러운 말투를 함께 기억합니다.

읽기와 듣기는 자연스러운 영어를 알아보는 힘을 만들고, 말하기와 영작은 뜻과 상황에서 영어를 직접 꺼내는 힘을 만듭니다. 같은 문장을 양쪽에서 연습하면 “본 적 있는 표현”이 “내가 쓸 수 있는 표현”으로 이어집니다.

03 · 첫 학습 루프

문장카드 → 카드 담기 → 복습 → 영작

왼쪽 메뉴에서 아래 네 곳만 기억하면 첫 학습 루프를 끝낼 수 있습니다.

읽고 듣기 · Input → 웹 리더 → 복습 → 말하고 쓰기 · Output → 영작 훈련

조작만 먼저 익히고 싶다면 관리 → 튜토리얼 → 읽기 카드 → 웹 리더의 샌드박스를 사용하세요. 튜토리얼 진행률만 기록되고 실제 카드 목록은 바뀌지 않습니다.

1) 웹 리더에서 문장 발견하기

  1. 읽고 듣기 · Input → 웹 리더를 엽니다.
  2. 홈의 Language Miner 연습 글을 엽니다. 앱에 내장된 글이라 외부 웹사이트에 접속하지 않습니다.
  3. running a little late를 드래그하고 선택 도구의 문장카드를 누릅니다.

2) 읽기 카드 확인하고 담기

미리보기에서 원문, 자연스러운 뜻, 선택 표현과 가까운 문맥을 확인합니다. 자동으로 채워진 내용도 직접 고칠 수 있습니다. 사람 이름, 연락처, 업무 내용처럼 남기고 싶지 않은 정보는 지운 뒤 카드 담기를 누르세요.

기본 샘플과 같은 문장입니다. 새 프로필에 같은 문장의 샘플 카드가 있을 수 있어, 연습 글에서 저장하면 유사한 카드가 둘 보일 수 있습니다. 흐름을 확인한 뒤 필요 없는 하나를 숨기거나 삭제하세요.

3) 첫 복습에서 먼저 떠올리기

  1. 복습을 열고 읽기 카드 덱의 복습 시작을 누릅니다.
  2. 앞면을 보고 뜻과 상황을 먼저 떠올립니다.
  3. 답 보기를 누른 뒤 실제 기억 상태에 맞게 다시, 어려움, 좋음, 쉬움 중 하나를 고릅니다.
평가는 점수가 아닙니다. 다음에 언제 다시 볼지 정하는 신호입니다. 답을 보고 나서 기억났다면 낮은 평가가 더 도움이 될 수 있습니다.

4) 같은 뜻을 보고 영어로 직접 쓰기

  1. 말하고 쓰기 · Output → 영작 훈련을 엽니다.
  2. “친구에게 조금 늦는다고 알려 주세요”라는 상황을 보고 I’m running a little late.를 직접 입력합니다.
  3. 답 보기로 내 답안과 추천 답안을 비교합니다. 다시 해 보고 싶으면 다시 쓰기를 누릅니다.

5) 캐릭터 대화에서 써 보기 — 선택

캐릭터챗은 AI 연결이 필요한 선택 기능입니다. 말하고 쓰기 · Output → 캐릭터챗에서 캐릭터와 학습 언어로 실전 연습을 고른 뒤 같은 문장을 보내 보세요. 클라우드 AI라면 시작 전에 캐릭터 설정, 최근 대화, 내 메시지와 카드 힌트가 외부로 전송될 수 있다는 안내를 확인합니다.

04 · 카드 세 종류

막혔던 방식에 맞는 카드만 만들면 됩니다.

카드만드는 곳앞면에서 하는 일
읽기 카드웹 리더, 문서 리더기영어를 보고 뜻과 구조 떠올리기
듣기 카드듣기 루프, 영상 리더원본 소리나 구간을 듣고 문장 알아보기
말하기 카드라이프 마이닝모국어 상황에서 학습 언어로 표현 꺼내기

세 종류를 한 화면에서 동시에 만드는 구조가 아닙니다. 글로 보면 아는데 소리에서 놓쳤다면 듣기 카드, 평소 자주 하는 말을 영어로 바로 꺼내고 싶다면 말하기 카드를 추가하세요.

05 · 내 자료로 확장

읽던 문서와 듣던 소리에서 바로 재료를 고릅니다.

문서 리더기

PDF나 지원 문서를 열고 표현을 선택해 읽기 카드로 만듭니다. 북마크와 최근 문서에서 다시 이어 봅니다.

듣기 루프

자막 구간을 반복하고 잘 안 들리는 소리 덩어리를 표시해 듣기 카드로 저장합니다.

영상 리더

자막 또는 전사 구간과 반복 범위를 확인한 뒤 현재 구간을 듣기 카드로 저장합니다.

라이프 마이닝

실제로 말하고 싶었던 모국어 표현을 말하기 카드 후보로 모읍니다. 자동 수집은 기본적으로 꺼져 있습니다.

미디어 백업 범위 카드와 전사 기록은 백업할 수 있지만 원본 PDF·오디오·영상과 로컬 파일 경로는 포함되지 않습니다. 다른 PC에서 복원하면 원본을 다시 선택해야 할 수 있습니다.

06 · PlayZone

학습 뒤에 다시 돌아오고 싶게 만드는 놀이 공간입니다.

캐릭터챗이 표현을 대화에서 시험하는 곳이라면, PlayZone은 학습으로 얻은 로컬 보상과 표현을 게임에 연결하는 곳입니다. 공식 게임도 앱 용량을 줄이기 위해 처음 사용할 때 내려받을 수 있습니다.

  1. 플레이존에서 어비스 서머너, 드릴하트 디펜스, 고양이 오디세이 중 하나를 고릅니다.
  2. 설치하고 플레이를 누릅니다.
  3. 다운로드 용량, 제작자, 버전, 원본, 라이선스, SHA-256과 저장·네트워크·다이아 권한을 확인합니다.
  4. 확인하고 다운로드를 누릅니다. 기술 검사가 끝나 실행 가능한 상태가 되면 플레이를 누릅니다.
외부 UGC의 범위 앱은 파일 구조, 무결성과 요청 권한을 기술적으로 검사합니다. 콘텐츠의 적절성, 저작권이나 제작자 주장까지 보증하지는 않으므로 출처와 라이선스를 직접 확인하세요.

07 · AI 연결

필요한 기능에만, 경계를 확인하고 연결합니다.

관리 → 설정 → AI · 음성 → 카드 생성

AI 미연결

새 설치의 기본 상태입니다. 카드 관리와 복습 등 AI가 필요 없는 기능부터 사용할 수 있습니다.

Ollama

별도로 설치한 로컬 모델을 사용합니다. localhost, 127.0.0.1, ::1 같은 loopback 주소만 로컬로 표시됩니다.

Gemini · Google

본인의 키를 사용하는 BYOK 방식입니다. 공급자, 모델, 전송 내용, 예상 호출과 비용을 확인한 뒤 실행합니다.

ChatGPT 웹

프롬프트를 직접 복사하고 결과 JSON을 붙여넣는 수동 방식입니다. 앱은 로그인 쿠키나 ChatGPT 페이지를 읽지 않습니다.

08 · 백업과 데이터 삭제

백업에는 학습 기록을, 키와 원본 파일은 제외합니다.

관리 → 설정 → 동기화 · 데이터 → 전체 백업·복원

  • 포함: 카드, 복습, 프로필, Life Mining, 듣기 기록, 캐릭터 대화, 루틴·미션, PlayZone 저장 등
  • 제외: API 키, 로그인 쿠키, 로컬 원본 경로, 로그, 캐시, 모델, 원본 PDF·오디오·영상

백업 파일 만들기로 .lembackup 파일을 저장합니다. 복원할 때는 체크섬과 버전, 충돌을 확인한 뒤 새 프로필로 복원, 겹치지 않는 데이터 합치기, 현재 데이터 교체 중 결과를 미리 보고 고릅니다.

09 · 브라우저 확장 베타

허용할 사이트와 문맥 범위를 먼저 줄입니다.

첫 공개 베타의 확장은 Chrome Web Store가 아닌 선택형 수동 설치입니다. 공식 Release에서 Language-Miner-Extension-<version>.zip과 SHA256SUMS.txt를 받아 해시를 확인하고 압축을 푸세요. Chrome의 chrome://extensions에서 개발자 모드를 켜고 압축 해제된 확장 프로그램을 로드한 뒤, manifest.json이 바로 보이는 폴더를 선택합니다.

  • 허용 사이트와 본인 메시지만 수집할지 선택합니다.
  • 앞뒤 문맥, 최대 길이와 마스킹 규칙을 정합니다.
  • 텍스트를 드래그하는 것만으로는 전송하지 않습니다. 문장 카드 동작을 실행한 뒤에만 선택 표현과 가까운 문맥을 로컬 앱으로 보냅니다.
  • 전달 실패 항목은 오프라인 대기 큐에서 확인하고 모두 지울 수 있습니다.

10 · 막혔을 때

오류 문구보다 먼저 상태와 다음 행동을 확인하세요.

Ollama가 연결되지 않음
Ollama 설치·실행 상태, 내려받은 모델명, loopback 주소를 확인한 뒤 Ollama 실행 및 연결을 다시 누릅니다.
게임팩을 계속 불러오는 중
공식 게임의 설치 상태와 실행 전 보안 리포트를 확인합니다. 기술 세부 정보는 펼쳤을 때만 복사해 이슈에 첨부하세요.
복원 뒤 미디어가 열리지 않음
원본 PDF·오디오·영상과 경로는 백업되지 않습니다. 원본 파일을 다시 선택하세요.
SmartScreen 경고
공식 Release와 SHA-256을 모두 확인하지 못했다면 실행하지 마세요.

이제 무엇을 해 볼까요?

읽기 카드 하나를 더 만들거나, 소리에서 놓친 문장을 듣기 카드로 저장해 보세요. 학습 재료는 많을 필요보다 실제로 다시 쓸 수 있는지가 중요합니다.
